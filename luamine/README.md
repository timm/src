# luamine

AI primitives (Num, Sym, dist, tree) and apps (cluster, acquire, GA/DE/SA) in Lua.
Conventions: [../etc/style.md](../etc/style.md).

Three requireable modules assembled from page-sized topic
files: `lib` (helpers), `luamine` (primitives), `lapps`
(apps). The ten-lecture REPL course: [tut.md](tut.md).

## Install

    curl -fL https://raw.githubusercontent.com/timm/src/refs/heads/main/luamine/INSTALL.md | sh
    lua luamine-eg.lua --all   # tests
    lua luamine-eg.lua -h      # options, demos

Data comes from [moot](https://github.com/timm/moot)
(`$MOOT` env var, else `~/gits/moot`).
