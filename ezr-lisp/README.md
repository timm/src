# ezr-lisp

Kin: [ezr-py](../ezr-py/) | [ezr-lua](../ezr-lua/);
course ancestor [attic/luamine](../attic/luamine/).

Same landscape ideas in Common Lisp, tiny-function style.
Explainable multi-objective optimization: sample a data
landscape under a small labelling budget, grow a regression
tree over the labels, pick good rows, and show which
x-ranges explain them.

Little reports live beside the code: the RQ0-RQ2 study is
[REPORT.md](REPORT.md). Conventions: [STYLE.md](STYLE.md).

## Install

    curl -fL https://raw.githubusercontent.com/timm/src/refs/heads/main/ezr-lisp/INSTALL.md | sh
    sbcl --script xai-eg.lisp --all   # tests, ~0.1s
    sbcl --script xai-eg.lisp -h      # options, tests

Data comes from [moot](https://github.com/timm/moot)
(`$MOOT` env var, else `~/gits/moot`).
