# ezr-lua

Kin: [ezr-py](../ezr-py/) | [ezr-lisp](../ezr-lisp/);
course ancestor [attic/luamine](../attic/luamine/).

Explainable multi-objective optimization: label a few rows,
let a tree explain the rest (Lua). A 14-lesson course whose
asserts are its tests; concepts live in the shared
[glossary](../glossary.md).
Conventions: [../etc/style.md](../etc/style.md).

## Install

    curl -fL https://raw.githubusercontent.com/timm/src/refs/heads/main/ezr-lua/INSTALL.md | sh
    cd ezr-lua && lua xai-eg.lua --all    # tests
    lua xai-eg.lua -h                 # options, sections, tests

Data comes from [moot](https://github.com/timm/moot)
(`$MOOT` env var, else `~/gits/moot`).
