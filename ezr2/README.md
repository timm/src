# ezr2

Explainable multi-objective optimization: label few rows, tree explains the rest (Python).
Study (RQ0-RQ2 over the moot corpus): [REPORT.md](REPORT.md).
Conventions: [../etc/style.md](../etc/style.md).

## Install

    curl -fL https://raw.githubusercontent.com/timm/src/refs/heads/main/ezr2/INSTALL.md | sh
    python3 ezr2-eg.py all    # tests
    python3 ezr2-eg.py -h     # options, tests

Data comes from [moot](https://github.com/timm/moot)
(`$MOOT` env var, else `~/gits/moot`).
